# Projeto_AV2_NatDiscreta
Projeto desenvolvido para a AV2 da disciplina de Resolução de Problemas de Natureza Discreta, com o objetivo de criar um jogo educativo sobre Progressão Geométrica (PG). O sistema funciona como um quiz interativo com feedback imediato, auxiliando alunos do ensino médio na compreensão de conceitos como razão, termos e padrões de sequências.
